# Heap and Heap Sort (COMP8090SEF Project Task 2)

In this repository, it contains a simple implementation of a **min-heap** data structure and the **heap sort** algorithm in Python.

## 1. Files

- `heap.py`: definition of the `MinHeap` class (array-based min-heap).
- `heapsort.py`: implementation of the `heap_sort` function using `MinHeap`.
- `examples.py`: simple command-line examples and test cases.

## 2. How to run

### 2.1 Running the demos

From the project directory:

```bash
python examples.py
